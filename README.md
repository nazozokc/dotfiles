# dotfiles

nazozokc の個人用 dotfiles リポジトリ。  
複数環境で同じ開発・ターミナル環境を再現するための設定ファイル集。

## 管理しているもの

- **Neovim** (`nvim`)
- **fish shell** (`fish`)
- **zsh** (`zsh`)
- **WezTerm** (`wezterm`)
- **Nix** (`nix`)

基本的に Linux 環境向け。  
コピペ運用は非推奨。

---

## 前提

以下は自分で入れておくこと。

- `git`
- `neovim`
- `fish` または `zsh`
- `wezterm`
- （`nix` ディレクトリを使うなら）`nix`
